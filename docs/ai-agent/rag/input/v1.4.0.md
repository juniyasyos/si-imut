# Release v1.4.0

**Tanggal**: 2026-06-13

## Ringkasan

Rilis besar setelah 629 commits sejak v1.3.4. Fokus utama pada **refactor arsitektur modular monolith**, **optimasi performa Daily Report**, dan **integrasi IAM/SSO**.

---

## Added

### Modular Monolith
- Refactor arsitektur ke modular monolith dengan 7 modul independen: Authorization, Benchmarking, DailyReport, FormEngine, ImutMaster, Laporan, Reporting.
- Setiap modul memiliki struktur internal konsisten (Contracts, Database, Filament, Http, Livewire, Models, Policies, Services, Providers).
- Service layer dipisahkan per modul dengan interface contracts.

### Daily Report & Livewire
- Server-side pagination dan search filter untuk Daily Report.
- Matrix snapshot synchronization dengan polling interval reduction.
- Date legend component untuk mobile navigation.
- Repository Pattern dengan Unified Compliance Service.
- Composite index pada `(form_template_id, report_date)` untuk optimasi query.

### Form Engine & Benchmarking
- Form template versioning dengan validity window.
- SyncFormTemplateDates command untuk sinkronisasi tanggal.
- Compliance scoring logic dengan role-based recommendations.
- Phase 4 Benchmark and Consolidation Tests.

### IAM/SSO & Auth
- Integrasi `juniyasyos/nexaid-client` menggantikan `auth-bridge-client`.
- IAM Application Switcher Component.
- LogoutController untuk menangani SSO dan local logout.
- IAM session expiration fix.
- Dynamic data permissions berdasarkan user authentication context.

### Backup & Export
- Integrasi `juniyasyos/filament-backup` v3.0.1.
- Backup configuration table dengan kolom baru.
- Export JSON data Unit Kerja.
- TestTtdUrlResolver command.
- Database dump configuration (timeout, SSL, custom connection).

### Licensing & Compliance
- Proprietary license resmi.
- SBOM license audit workflow.
- Third-party licenses documentation.
- LICENSE file resmi.

### UI & Frontend
- Dashboard components: state management, loading indicators.
- Daily report entry resources refactor.
- Asset management: old theme assets dihapus, CSS compressed, manifest update.
- Custom scrollbar styles dan scroll tip.
- Compliance display component.

### Dependencies

- auth-bridge-client → nexaid-client.
- filament-backup → v3.0.1.
- manage-unit-kerja → v2.0.0.

## Fixed

### Performance
- 30-second loading bottleneck di daily report dashboard — matrixData payload serialization dihapus.
- N+1 query imut penilaian — mengurangi koneksi dari 5000+ menjadi normal.
- Load indicator lebih cepat dengan cache.

### Livewire
- Nested Alpine x-data menyebabkan Uncaught ReferenceError.
- Reset pagination ke 1 saat selected date berubah.
- Remove debug pooling update via AJAX.
- Separate loading states untuk date nav dan main content.

### Bugs
- Indicator not found saat filter search di daily reports.
- Typo di notification body untuk report calculation.
- Disabled state logic di buildAnalysisSchemaForAction.
- ReadOnly → disabled untuk analysis input fields.
- Filter action di header list daily report.

## Breaking Changes

- **auth-bridge-client dihapus**: Semua komponen yang menggunakan auth-bridge-client harus migrasi ke nexaid-client.
- **ToggleIconColumn dihapus**: Gunakan IconColumn sebagai pengganti.
- **FormBuilder pages dihapus**: Form template management via FormTemplateVersionsRelationManager.
- **ImutReportService namespace berubah**: Dari `App\Services\Laporan\ImutReportService` → `App\Modules\Reporting\Services\ReportingService`.

## Migration Notes

1. Update `.env` dengan konfigurasi nexaid-client (IAM):
   ```ini
   IAM_BASE_URL=
   IAM_CLIENT_ID=
   IAM_CLIENT_SECRET=
   ```
2. Jalankan migrasi database:
   ```bash
   php artisan migrate
   ```
3. Generate ulang permission:
   ```bash
   php artisan shield:generate --all --panel=admin
   ```
4. Update dependency:
   ```bash
   composer update
   npm install && npm run build
   ```
5. Hapus cache:
   ```bash
   php artisan optimize:clear
   php artisan config:cache
   ```
