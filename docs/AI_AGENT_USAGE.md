# AI Agent Usage for SIIMUT

Panduan utama untuk AI Agent saat memproses repository SIIMUT:

1. **Jika task tentang SIIMUT**, pakai `bunx contexta` sebagai peta awal.
2. **Jangan baca seluruh repo** kalau hasil `contexta` cukup.
3. **Kalau `contexta` kurang jelas/gagal**, baru baca source file relevan.
4. **Source code tetap source of truth**.

Selalu manfaatkan `contexta` untuk mengekstrak dan menavigasi arsitektur SIIMUT.
