# Mobile Components

## Components

### `mobile-indicator-card.blade.php`
Mobile-optimized view untuk indicator cards.

**Features:**
- Compact card layout
- Touch-friendly buttons
- Responsive design
- Essential information only

### `mobile-action-buttons.blade.php`
Mobile action buttons dengan optimized touch targets.

**Features:**
- Large touch targets
- Icon + text labels
- Responsive spacing

### `mobile-card.blade.php`
Generic mobile card layout component.

**Features:**
- Consistent card styling
- Mobile-optimized spacing
- Dark mode support

### `mobile-status-cards.blade.php`
Mobile status cards untuk quick status overview.

**Features:**
- Status summary cards
- Color-coded indicators
- Swipeable layout

**Usage:**
```php
@include('filament.resources.daily-report-entry-resource.pages.partials.components.mobile.mobile-indicator-card')
```
